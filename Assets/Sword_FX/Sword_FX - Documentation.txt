Load "Sword_FX.unity" for an example scene, or simply drag the required sword from the Prefab folder (in the Project window) into the Hierarchy window.

To avoid sorting issues with the particles often drawing behind the sword blade, it is advisable to turn off Soft Particles. This setting can be found under the menu Edit -> Project Settings -> Quality
